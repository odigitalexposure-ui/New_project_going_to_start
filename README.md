# Stay Homes

Premium homestay and rental property website built with Next.js App Router, TypeScript, and Tailwind CSS.

## Stack

- Next.js
- TypeScript
- Tailwind CSS
- Static content architecture

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Notes

- The contact form is implemented as a static front-end experience and can be connected to any backend later.
